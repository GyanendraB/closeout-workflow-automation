# Closeout Workflow Automation

The runnable Playwright project lives in `closeout-ai-validation-automation`.

Use the root scripts below so commands are forwarded to the correct project and Playwright instance:

```bash
npm test
npm run test:ui
npm run test:api
npm run report
```

If you prefer to run commands manually, first change into `closeout-ai-validation-automation` and then run Playwright from there.
